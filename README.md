# LinguaScience — Přírodopis 6. ročník

Interaktivní webová učebnice ve stylu Lingua brand (krémové pozadí, Fraunces serif, modrý italic accent).

## Struktura souborů

```
index.html        ← landing page (sedm kapitol)
kapitola1.html    ← Od vesmíru k Zemi
style.css         ← design systém
app.js            ← Quiz · True/False · Fill-in · Leitner CLIL
.nojekyll         ← pojistka proti Jekyll buildu
```

Vše v rootu — žádné podsložky.

## Co je hotové

- Landing s logem Lingua, dvěma oblastmi (Vesmír a Země / Život na Zemi)
- Kapitola 1 kompletní — výklad, 30 CLIL kartiček, testík (3 části), zdroje

## Brzy

Kapitoly 2–7 čekají na výchozí texty.
